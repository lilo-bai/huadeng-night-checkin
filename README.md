# 华灯之夜次元庆典 · 电子签到板

一个国风与二次元风格结合的电子签到板网页应用。

## 功能特性

- 📸 **图片上传**：支持点击或拖拽上传照片，自动压缩优化
- ✍️ **名字填写**：输入你的名字或昵称留下印记
- 🎠 **跑马灯展示**：签到内容以跑马灯形式循环展示
- 🖼️ **全员集结**：网格形式展示所有签到嘉宾
- 💾 **本地存储**：使用 LocalStorage 保存签到数据
- 📱 **响应式设计**：适配手机、平板、电脑等各种设备

## 设计风格

- 🏮 国风元素：灯笼、云纹、传统配色（朱红、金色、墨色）
- 🎭 二次元风格：明亮渐变、可爱装饰、发光效果
- ✨ 明亮主题：暖白背景，温馨舒适
